# InfoEdge Roadmap

InfoEdge is early-stage open source software. This roadmap keeps the next maintenance work visible and easy to discuss in issues and pull requests.

## v0.1.x

- Improve the source registry with clearer public, gated, third-party, and restricted source states.
- Add more deterministic tests for connector payload normalization.
- Document the minimal local backend setup for PostgreSQL and Redis.
- Add screenshots for major workbench views.
- Keep API acceptance scripts stable as the dashboard evolves.

## v0.2.x

- Add connector health reporting and last-refresh metadata.
- Add a safer configuration flow for authorized sources.
- Introduce release notes for connector additions and breaking schema changes.
- Expand CI to cover linting and frontend acceptance smoke checks.

## Contribution Areas

- Public source connectors.
- FastAPI endpoint coverage.
- React dashboard usability.
- Documentation and examples.
- Security and data-source compliance review.
